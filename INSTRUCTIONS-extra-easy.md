# משימה: תרגום extra-easy לבראשית 1-15

## רקע
הוספנו רמת תרגום חדשה — "פשוט מאוד" (extra-easy) — שמיועדת לילד בן 10 בלי שום רקע. הפרומפט קיים ב-`translation-prompts/extra-easy.txt`. ה-UI כבר עודכן בקוד לתמוך ב-5 רמות, חסרים רק קבצי התרגום עצמם.

## מה לעשות

1. קרא את `translation-prompts/extra-easy.txt` ולמד את הסגנון. זה דרגה אחת יותר נמוכה מ-`easy.txt` — אפילו "עתה" הופך ל-"עכשיו", "אנכי" ל-"אני", "ויאמר" ל-"אמר", וגניאלוגיות הופכות לסיפור רגיל.

2. לכל פרק 1-15 בבראשית:
   - הורד את העברית עם נקודות (בלי טעמים) מ-Sefaria:
     ```
     https://www.sefaria.org/api/v3/texts/Genesis.{N}?version=hebrew
     ```
     נקה: הסר תגי `<...>`, הסר `׃` בסוף, הסר טעמי מקרא (יוניקוד `\u0591-\u05AF`, `\u05BD`, `\u05C0`, `\u05C3`, `\u05C6`) — השאר את הניקוד.
   
   - שלח לתרגום עם הפרומפט `extra-easy.txt` והפסוקים מסומנים `[0] verse`, `[1] verse`...
   
   - שמור ב: `public/translations/Genesis_{N}.extra-easy.json` במבנה:
     ```json
     {
       "0": "פסוק 0 בעברית פשוטה מאוד",
       "1": "פסוק 1 בעברית פשוטה מאוד"
     }
     ```

3. **ולידציה לכל פרק:** מספר המפתחות חייב לתאום למספר הפסוקים שהורדו. אם לא — הרץ שוב.

4. **סה"כ ייווצרו 15 קבצים חדשים.**

## אסור לגעת בקבצים אחרים
- אל תיגע ב-`Genesis_*.easy.json`, `Genesis_*.medium.json`, `Genesis_*.close.json` הקיימים.
- אל תיגע ב-`.no-nikud.json`.
- אל תיגע בפרקים 16-50.
